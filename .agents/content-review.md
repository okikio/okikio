# Content review

Use this note when a task changes visible copy, page structure, or portfolio presentation.

Check for:
- factual accuracy against the repository's existing content
- consistent personal voice and tone
- accessible headings, link text, image alt text, and readable hierarchy
- concise copy that still sounds like the site owner

If a stronger wording change would require information not present in the repository, raise it as an open question instead of inventing it.
